A simple and customizable accordion block for the WordPress Gutenberg editor.

== Description ==

The Accordion Block plugin adds a flexible and interactive accordion block to the WordPress block editor (Gutenberg). 

### Features:
- **Gutenberg-native**: Easily insert accordions directly within the block editor.
- **Collapsible Sections**: Each accordion item expands and collapses with a simple click.
- **Custom Titles**: Edit accordion titles from the block settings panel.
- **Inner Blocks Support**: Nest any other blocks inside the accordion content.
- **Fully Responsive**: Works across all devices and screen sizes.
- **Customizable**: Supports CSS styling and additional JavaScript functionality.

This plugin is ideal for creating **FAQs, collapsible sections, and structured content layouts**.

== Installation ==

To install this plugin:

1. Upload the plugin files to the `/wp-content/plugins/accordion-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. In the **Gutenberg editor**, search for **Accordion Block**, insert it into your post or page, and start adding content.
4. Customize the block via the settings panel in the editor.

== Frequently Asked Questions ==

= How do I add an accordion to my post? =

Simply insert the **Accordion Block** from the block editor, set a title, and add content inside the accordion.

= Can I have multiple accordion blocks on the same page? =

Yes! You can add as many accordions as needed, and each one will function independently.

= Does this work with all themes? =

Yes, the block is designed to work with any Gutenberg-compatible theme. However, you may need to adjust styles for perfect integration.

= Can I change the default open/closed state of the accordion? =

Yes, you can set whether the accordion starts open or closed from the block's settings panel.
